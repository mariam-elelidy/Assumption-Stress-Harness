CROSS-SCENARIO PATTERNS
──────────────────────────────────────────────────────────────────────────────

Pattern 1 — Silent failures (width unchanged, coverage collapsed):
  Label Noise Shift and Concept Drift both produce coverage ~0.20-0.30 while
  maintaining nearly identical width to baseline (2.053 vs 2.054). An operator
  monitoring interval width would see nothing wrong. Coverage would only be
  detected by comparing against a held-out ground-truth set — which is rarely
  done post-deployment.

Pattern 2 — The temporal failure is the worst:
  Temporal ordering produced the lowest coverage (0.126) AND the largest width
  (48.3) simultaneously — the worst possible outcome. This is because the trend
  inflates calibration residuals (making q large), but test residuals are even
  larger (making coverage poor). A deployment system relying on these intervals
  would be both useless and confidently wrong.

Pattern 3 — Outlier inflation vs small cal set have opposite mechanisms but
  similar symptoms (over-coverage):
  - Heavy tails: q inflated by outlier residuals → width +50%
  - Small cal set: q inflated by coarse quantile grid → width +17%, variance +13×
  The practical risk is different: heavy tails produce systematically uninformative
  intervals; small cal sets produce intervals whose reliability cannot be trusted
  run-to-run even if the average looks fine.

Pattern 4 — Covariate shift is recoverable:
  A 3-unit shift in X caused only -0.032 coverage damage — recoverable with
  weighted conformal prediction (Tibshirani et al., 2019). This is the one
  scenario where an extension (importance weighting) directly fixes the problem.
